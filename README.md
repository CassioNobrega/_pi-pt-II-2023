### Requisitos

### 1 - passo | Baixar Repositório
- `git clone https://github.com/CassioNobrega/_pi-pt-II-2023`

### 2 - passo | Entrar na Pasta
- `cd _pi-pt-II-2023/`

### 3 - passo | Instalar Dependências
- `npm install`

### 4 - passo | Iniciar Aplicação
- `npm start`
